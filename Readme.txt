Weather Application - User Guide

Introduction:
Welcome to the Weather Application! This simple web app allows you to check the current weather conditions for a location of your choice. Whether you want to plan your day or a trip, our app provides you with up-to-date weather information.

Getting Started:

1. Homepage: When you open the app, you'll see a user-friendly homepage with the following elements:
A search input field where you can enter a city name or ZIP code to specify the location for which you want to check the weather.
A "Get Weather" button that triggers the weather request.
A dropdown menu to select your preferred temperature unit (Celsius or Fahrenheit).
Using the App:
2. Search for Weather:

Enter the name of a city or a ZIP code in the search input field.
Choose your preferred temperature unit from the dropdown menu.
Click the "Get Weather" button.
3. Viewing Weather Data:

Once you click the "Get Weather" button, the app will fetch and display the following weather data for the specified location:
City name
Temperature (in your selected unit)
Humidity percentage
Wind speed (in kilometers per hour)
Weather description
Weather icon
4. Error Handling:

If you enter an invalid location, the app will display an error message (e.g., "City not found") and guide you on how to proceed.
If there is an issue with the API request, you will see an error message indicating that an error occurred while fetching data.
5. Switching Temperature Units:

You can easily switch between temperature units (Celsius and Fahrenheit) using the dropdown menu.
The displayed temperature will update according to your selection


Tips:

*Ensure you have an active internet connection to fetch weather data.
*If you encounter any issues, check the error messages for guidance.
*Please be patient as the app fetches the latest weather data from the API.
